Overview

This UI toolkit is a highly-modular design system for rapid web page development. It contains different materials that can be assembled into more complex page layouts.

This guide contains real working examples, code snippets, documentation, and style guidelines.

Usage

This project is configured to compile it's stylesheets and javascript into a cache busted bundle and injected into the static html.

The production build script will output each page as a .cshtml partial wrapped in the partials.cshtml layout found at ./src/views/layouts/partials.cshtml

Taxonomy

Elements

Single html tags with one to many states or styles. Can be used in components, views, or templates.

Components

A grouping of elements in a portable collection that can contain self-instantiating JavaScript.

Views

Group together components and elements into a unique portable component.

Containers

Contain modules, components, and elements and provide structure for the page.

Templates

Pieces containers together for a full-page structure.

Pages

Pages are specific instances of templates that show what a UI looks like with real representative content in place.
